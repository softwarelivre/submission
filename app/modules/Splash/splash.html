SEGUE

O SEGUE - Sistema Extensível de Gerenciamento Unificado de Eventos é um projeto apoiado pela Associação Software Livre.Org, iniciado em 2015. Está disponibilizado no GitHub para que a comunidade possa acompanhar e contribuir com seu desenvolvimento. O SEGUE é o principal sistema utilizado para realizar as inscrições e o temário do FISL e está a disposição para o apoio a qualquer evento de software livre no país e no mundo.
